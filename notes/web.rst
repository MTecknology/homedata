Website Development
===================

Most of the documentation I wrote on managing websites can be found at:
   https://handbook.recoverysource.net/.

General Requirements
--------------------

Any website should try to **meet these design goals**.

- 4th Grade Reading Level (Prefer Lower)
- Focus is Newcomer
- All 12-Step Groups
- All-in-One Front Page
- No Javascript Bloat
- Separate FAQ Page for Obscure Information

  + Anything must be linked to from a relevant front-page section

- Keep it Simple
- Answer as many questions as possible in a few short sentences.

  + It's about ability to digest information, not getting all possible questions answered

**Front Page Questions**:

- Part One:

  + What is this site?
  + Why do I care?
  + What can it do for me?
  + How much do I have to pay?
  + Do I actually belong here? (or can this help someone that I know?)

- Part Two:

  + Who controls this site?
  + What if information is wrong?
