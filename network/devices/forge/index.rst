Debian Forge
============

Vagrant (temporary) workstation for building and testing Debian packages.

Common Usage:

- ``cd ~/repos/data/network/devices/forge``

  ===============  ============
  Command          Accomplishes
  ===============  ============
  vagrant up       Spin up vagrant VM and run all initial setup
  vagrant ssh      Log in to vagrant VM
  vagrant down     Stop and destroy vagrant VM
  vagrant destroy  Destroy vagrant VM
  ===============  ============

- ``~/.sync`` allows copying files in to / out of  vagrant VM
