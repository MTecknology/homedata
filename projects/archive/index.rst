Other Projects
==============

These are various projects that I have built over the years. Some have been for
companies, others were built for fun.

Note: Where required, release information has been obtained from the
appropriate (company) copyright holder.

Repository List:

- `Tiny Dockable Clock <https://github.com/MTecknology/tdc>`__:
   Tiny and lightweight clock designed to fit docks and system trays
- `Device42 Command Line <https://github.com/MTecknology/d42-cli>`__:
   Command line interface for Device42
- `Static Swagger-UI <https://github.com/MTecknology/static_swagger-ui>`__:
   Read-only web page similar to Swagger-UI
- `Pelican Search <https://github.com/MTecknology/pelican-search>`__:
   Lightning-fast searching to otherwise-static pelican websites
- `Collapsible 3D Rope <https://github.com/MTecknology/rope-demo>`__:
   Quick demo of a rope that disappears into the ceiling 
- `User-Respecting Web Theme <https://github.com/MTecknology/user-theme>`__:
   Very basic demonstration showing respect for user's theme preference
- `Debian Development Toolbox <https://github.com/MTecknology/debdev>`__:
   An attempt to make debian development easier to learn
- `Debian SSO <https://github.com/MTecknology/debian-sso>`__:
   Plugin for Lemonldap::NG to support authentication using Certificates
- `Script-Helpers <https://github.com/MTecknology/script-helpers>`__:
   A simple set of helper functions to aid the development of scripts
- `Linux Checker <https://github.com/MTecknology/lincheck>`__:
   Proof of concept to evaluate custom compliance checkers
- `DFSG-Review <https://github.com/MTecknology/dfsg-review>`__:
   DCheck helps Debian Developers review packages for DFSG (and Policy) compliance
- `LaTeX Resume <https://github.com/MTecknology/latex-resume>`__:
   Template to Build Resume (Summary) and Curriculum Vitae (Course of Life) using LaTeX

   + https://mtecknology.github.io/latex-resume/
   + https://www.overleaf.com/latex/templates/mtecks-resume/fzgztpkgngjc

- `Teckhost <https://github.com/MTecknology/teckhost>`__:
   Configuration management for personal and family devices

   + https://github.com/MTecknology/teckhost/tree/master/test/compliance: CIS Benchmark Tests

- `Disturbance Tracker <https://github.com/RTAAS/disturbance-tracker>`__:
   Enviromental monitor that trains a machine learning model to detect concerning noises

   + https://rtaas.github.io/disturbance-tracker/

- `Recovery Source <https://github.com/orgs/recoverysource>`__:
   Source code meant to help with 12th step work

   + https://handbook.recoverysource.net/: Handbook to teach beginners web stuff
   + https://sober.page/: Domain for all 12-Step recovery websites
   + https://github.com/recoverysource/aamod: Hugo theme to build an AA-focused website
   + https://github.com/recoverysource/hugo-website: Turn-key implementation of aamod
