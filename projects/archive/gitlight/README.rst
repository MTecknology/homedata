GitLight
========

**DEAD PROJECT**

Source Code:
  https://github.com/MTecknology/homedata/tree/master/projects/archive/gitlight

GitLight is a simple/lightweight collaboration tool built on top of Gitolite3.

Running GitLight
----------------

Prerequisites:

- `Gitolite3 Installed/Configured <https://docs.gitlight.io/walkthrough/gitolite>`__

GitLight can be run in either `production <https://docs.gitlight.io/walkthrough/deployment>`__
or `development <https://docs.gitlight.io/hacking/development>`__ modes.
